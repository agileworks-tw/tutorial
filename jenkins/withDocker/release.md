發佈專案使用 docker
-------------------

建置 image
----------

複製 task/[release](../task/release)

```
docker build --no-cache -t trunkworkshop/sailssample .
docker push trunkworkshop/sailssample:latest

docker rm -f sample
docker run -d -p 3333:1337 --link mysql --name sample trunkworkshop/sailssample
```
