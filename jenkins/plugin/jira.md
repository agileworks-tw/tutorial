commit message
--------------

JIRA plugin 將會根據你的 commit message 直接連結到相關的 JIRA issue

如下圖所示，若已有存在一個 issue 為 `TEST-1`

![](images/jira/commit_message.png)

則可以直接點選 link 進入檢視

------------
