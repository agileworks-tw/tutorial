branch preview
==============

將特定分支進行運行預覽，使用時機

1.	pull request merge 後確認 develop 分支執行狀況
2.	master 更新後確認執行狀況
3.	特定 hotfix 分支確認修正情形
4.	特定 feature 分支確認運作情形
5.	使用 fork 時，確認特定 user 相關程式運作的情形
