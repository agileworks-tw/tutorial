Build 自動建置
==============

使用時機
--------

1.	Daily Build
2.	週期性建置
3.	進行正式機部署時事先建置

Task 實作
---------

先設置 SVN Repository 使用：

```
http://localhost/svn/testrepo
```

### Shell 指令

```
#!/bin/bash
source ~/.bashrc
mvn clean
cp -r ~/node_modules ./ > /dev/null
mvn -DskipTests install package
```

第一次執行時，可以發現查無 Grunt 的問題，此時我們可以 login jenkins user

安裝 grunt-cli 透過：`npm i grunt-cli -g`

再一次執行，完成 build 建置
